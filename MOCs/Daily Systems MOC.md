## Systems Overview

### Daily Routine
1. **Morning**: Open daily note → set intentions → top 3 tasks
2. **Throughout day**: Capture to Inbox → quick notes
3. **Evening**: Reflection → link notes → review tasks

### Weekly Review (Sunday)
1. Review past week's daily notes
2. Process Inbox items
3. Update project statuses
4. Plan next week's priorities

### Monthly Review (1st of month)
1. Review weekly reviews
2. PARA audit — archive/activate projects
3. Knowledge garden review
4. Goals recalibration

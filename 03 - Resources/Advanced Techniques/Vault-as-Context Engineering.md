# Vault-as-Context Engineering

## Idea
Treat the vault itself as a context system: structure, metadata, and links determine what Claude can understand quickly.

## Design Principles
- Stable file locations
- Clear note types
- Rich but disciplined wikilinks
- MOCs for entry points
- Memory notes for continuity

## Rule of Thumb
If a human cannot navigate the vault easily, Claude will not reason over it efficiently either.

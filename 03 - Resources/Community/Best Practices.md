# Best Practices

## Core Practices
- Use templates for every new structured note.
- Prefer wikilinks over manual path references.
- Keep notes atomic when possible.
- Review and connect notes weekly.
- Archive aggressively when a project is done.

## Claude-Specific Practices
- Give Claude narrow, relevant context.
- Ask for concrete edits and outputs.
- Keep command files task-specific.
- Preserve human judgment for meaning and priority.

# Plugin Recommendations

## High Value Plugins
- Dataview for dashboards and queries
- Templater for dynamic note creation
- QuickAdd for fast capture and macros
- Calendar or Periodic Notes for time-based workflows
- Claudian for in-vault Claude workflows
- Obsidian Git for sync and backup discipline

## Choose Carefully
Prefer plugins with a clear workflow payoff, active maintenance, and minimal overlap with current tools.

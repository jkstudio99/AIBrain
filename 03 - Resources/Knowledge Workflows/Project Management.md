# Project Management

## Goal
Use the vault to keep project outcomes, decisions, tasks, and supporting knowledge in one place.

## Workflow
1. Create a project note in `01 - Projects/`.
2. Link reference notes and related knowledge.
3. Review status in daily and weekly systems.
4. Archive when outcomes are complete.

## Best Practice
Keep project notes outcome-focused while pushing deeper ideas into Resource or Knowledge notes.

# MOCs & Hub Notes

## Purpose
Maps of Content act as navigational hubs that gather key notes around one domain, workflow, or project.

## Good MOC Structure
- Short overview of the domain
- Curated links grouped by theme
- A dataview block for recent additions
- Links to adjacent MOCs

## When to Create a MOC
- A folder has more than 7-10 meaningful notes
- A topic spans multiple folders
- You want a stable entry point for Claude and for yourself

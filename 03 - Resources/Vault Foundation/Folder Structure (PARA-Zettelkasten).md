# Folder Structure (PARA-Zettelkasten)

## Model
The vault uses PARA for organization and Zettelkasten for idea development.

## PARA Roles
- `00 - Inbox/` for fast capture
- `01 - Projects/` for outcome-driven work
- `02 - Areas/` for ongoing responsibilities
- `03 - Resources/` for reference material
- `04 - Archive/` for inactive material

## Zettelkasten Roles
- `06 - Knowledge/Fleeting Notes/` for rough captures
- `06 - Knowledge/Literature Notes/` for source-based notes
- `06 - Knowledge/Evergreen Notes/` for atomic lasting ideas
- `MOCs/` for navigation and thematic hubs

## Placement Rules
- Put action-oriented work in Projects.
- Put systems and reference guides in Resources.
- Move finished projects to Archive.
- Promote strong ideas into Evergreen Notes.

# Team Collaboration

## Goal
Adapt the vault for shared use without losing note quality, metadata consistency, or ownership clarity.

## Requirements
- Shared naming rules
- Shared templates and command files
- Git or sync conventions
- Clear boundaries between personal and team notes

## Risk
Team collaboration increases entropy fast, so governance matters more than clever tooling.

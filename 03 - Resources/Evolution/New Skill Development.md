# New Skill Development

## Goal
Create focused Claude skills when a workflow is repeated often enough that examples and rules would improve output quality.

## Good Candidates
- Literature note extraction
- Research synthesis by domain
- Project retrospectives
- Vault health remediation

## Rule
Keep each skill narrow, example-rich, and easy to test.

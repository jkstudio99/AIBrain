Create a weekly synthesis by:

1. Reading all daily notes from the past 7 days
2. Reading all new notes created this week
3. Identifying recurring themes and patterns
4. Highlighting the most important insights
5. Creating a draft Weekly Review note using the template in `Templates/Weekly Review.md`
6. Saving to `05 - Daily Systems/Weekly Reviews/`
7. Suggesting priorities for next week based on patterns observed

Focus on patterns, growth, and actionable next steps.

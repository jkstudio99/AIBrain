## 💾 Hardware & Semiconductor

### 🔋 Tesla AI5 Chip — Milestone สำคัญ
- Elon Musk ประกาศบน X ว่า **Tesla AI5 chip** ถึง major engineering milestone แล้ว — กำลังเข้าสู่ production
- หุ้น Tesla **พุ่ง 8%** → ปิดที่ $391.95
- Tesla วางแผนสร้าง **"Terafab" 2 แห่ง** ในออสติน, เท็กซัส
  - โรงงาน 1: ชิปสำหรับรถยนต์และ humanoid robot
  - โรงงาน 2: orbital data centers (ร่วมกับ SpaceX)
- **Intel เข้าร่วมโปรเจคนี้**

### 🏭 TSMC — Q1 ทุบสถิติ
- รายได้ Q1 2026: **$35.7B** — เพิ่มขึ้น **35% YoY**
- ใกล้ขีดบนของ guidance ที่ตั้งไว้
- หุ้นปี 2026 ปรับขึ้นมาแล้ว **+20%**

### 🚀 Nuvacore — CPU Startup ใหม่จาก Ex-Apple
- **Gerard Williams** อดีตผู้บริหาร Apple chip (ออกแบบ A-series) ตั้ง **Nuvacore**
- เดิมพันว่า **CPU จะสำคัญขึ้นอีก** ในยุค AI
  - เหตุผล: inference, memory movement, orchestration ต้องการ CPU ที่ฉลาดขึ้น ไม่ใช่แค่ GPU
- เป็น challenger ของ Nvidia ในมุมแตกต่าง

### 🚗 Wayve (Self-Driving) — $60M จาก Qualcomm + AMD + Arm
- บริษัทรถยนต์ไร้คนขับจากอังกฤษ **Wayve** รับเงิน $60M
- ผู้ลงทุน: **Qualcomm, AMD, Arm** — ผู้ผลิตชิปชั้นนำทั้ง 3
- Approach พิเศษ: **mapless AI** — ไม่ต้องมี HD map ล่วงหน้า
- ท้าทาย Waymo และ Cruise

## 🔐 Cybersecurity — Patch Tuesday เดือนเมษายน

### Microsoft — Patch Tuesday (วันนี้)
> [!warning] ต้อง patch ด่วน
> - แก้ช่องโหว่ **160+ รายการ** รวม **SharePoint Zero-Day** ที่กำลังถูก exploit อยู่
> - **CVE-2026-32201** — CISA สั่งให้ federal agencies patch ภายใน **28 เม.ย.**

### Cisco — Critical Vulnerabilities
- แพตช์ช่องโหว่ **Critical 4 รายการ** ใน Identity Services + Webex
- ผลกระทบ: arbitrary code execution, user impersonation
- อัปเดต **ด่วนมาก** สำหรับองค์กรที่ใช้ Cisco

### Adobe + SAP — Patch Tuesday เดือนนี้
- **SAP Business Planning & Consolidation** — SQL Injection vulnerability อันตราย
- **Adobe** — หลายผลิตภัณฑ์ได้รับแพตช์

### ⚠️ PHANTOMPULSE RAT — ใช้ Obsidian เป็นทางเข้า
> [!danger] น่าสนใจมาก — เกี่ยวกับ Obsidian โดยตรง
> มีแคมเปญ social engineering ที่ **ใช้ Obsidian เป็น initial access vector** เพื่อแพร่กระจาย RAT ชื่อ **PHANTOMPULSE**
> - เป้าหมาย: บุคคลในวงการ **การเงินและ crypto**
> - วิธีการ: หลอกให้รัน vault หรือ plugin ที่มี malicious code
> - บทเรียน: **audit ทุก plugin และ vault ที่รับจากคนอื่น** ก่อนเปิด

## ☁️ Cloud & Enterprise IT

### Zero Trust เป็น Standard ใหม่แล้ว
- 68% ของ cloud breaches ในปี 2025 มาจาก **compromised service accounts และ forgotten API keys**
- ทุก org มี **40–50 automated credentials ต่อ 1 พนักงาน** (tokens, AI agent connections, OAuth)
- เทรนด์ 2026: **identity-based access control** + **AI-driven threat detection**

### IBM — Heavy Investment ใน AI Security
- IBM ลงทุนหนักใน AI-driven security systems ที่ detect และ respond แบบ real-time

## 📅 Daily Rhythm

### 🌅 Morning (09:00 - 10:00)

```
[ ] อ่าน morning brief (Telegram + 🏠 Home dashboard)
[ ] Review KPI dashboard (weekly metrics review output ล่าสุด)
[ ] Plan top-3 exec priorities วันนี้
[ ] `chief-of-staff` — 5-bullet brief (wins/blockers/priorities/risks/decisions)
```

### 🏃 Core Hours (10:00 - 17:00)

Skills ที่เรียกระหว่างทำงาน:
- `strategic-alignment` — Weekly OKR review + team alignment
- `scenario-war-room` — Big decisions (hire/fire, pricing, M&A, pivot)
- `decision-logger` — บันทึกทุก strategic decision
- `meeting-analyzer` — หลัง 1:1s + leadership sync
- `competitive-intel` — Market signal scan

### 🌆 EOD (17:00 - 17:30)

```
[ ] `changelog-generator` — สรุปวันนี้เป็น exec log
[ ] Review: any decision ที่ค้างต้องปิดคืนนี้?
[ ] Send EOD ping ถ้ามี blocker ต้องแก้ก่อนเช้าพรุ่งนี้
[ ] Update daily note ใน Obsidian
```

## 🧩 My Orchestration Patterns

### Pattern A — Strategic Decision (Big Bet)
**ใช้เมื่อ:** การตัดสินใจที่มี impact > 10% ของ revenue หรือ team
**Chain:**
```
1. `competitive-intel` → market context + trend
2. `scenario-war-room` → 3 scenarios (bull/base/bear) + pre-mortem
3. `financial-analyst` → cost/revenue impact
4. `strategic-alignment` → OKR fit check
5. Human decision → sign-off + `decision-logger` entry
```

### Pattern B — Monthly Board Prep
**ใช้เมื่อ:** End of month, preparing investor update
**Chain:**
```
1. `saas-metrics-coach` → metrics snapshot (ARR/MRR/churn/CAC/LTV)
2. `financial-analyst` → P&L + cash + burn
3. `product-analytics` → engagement + activation
4. `competitive-intel` → market update
5. `board-deck-builder` → deck structure + narrative
6. `strategic-alignment` → OKR status for next month
```

### Pattern C — Crisis / Incident Communication
**ใช้เมื่อ:** P0/P1 incident, PR crisis, customer escalation
**Chain:**
```
1. `incident-commander` → classify severity + comms plan
2. `chief-of-staff` → internal broadcast draft
3. Human review + send
4. `scenario-war-room` → post-incident strategic impact
5. `decision-logger` → what changed in strategy
```

## 🤝 Handoff Points

**รับงานจาก:**
- **CTO** → tech health summary (weekly), incident reports
- **Marketing Lead** → pipeline + campaign ROI (weekly)
- **บัญชี** → cash runway + P&L (monthly)
- **BA / Product** → discovery findings, PRD sign-off requests

**ส่งงานให้:**
- **Board** → monthly update (deck + narrative)
- **Leadership team** → weekly direction + OKR checkpoints
- **ทุกคน** → quarterly all-hands + vision refresh

**Review กับ:**
- **CTO** — weekly 1:1, joint incident review
- **Board** — monthly (full deck) + quarterly (strategic)
- **Leadership** — weekly staff meeting

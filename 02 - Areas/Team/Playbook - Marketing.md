## 🧩 My Orchestration Patterns

### Pattern A — Campaign Launch
**ใช้เมื่อ:** Launching new campaign (product / promo / event)
**Chain:**
```
1. `launch-strategy` → full plan (goals, channels, timeline)
2. `marketing-strategy-pmm` → positioning + messaging
3. `marketing-psychology` → buyer motivation
4. `content-creator` → 5 core pieces (blog + social + email)
5. `copywriting` → CTAs + subject lines
6. `email-sequence` → 5-touch drip
7. `paid-ads` → 3 ad variants
8. `social-content` → 10 social posts
9. `seo-audit` → landing page check (handoff to SEO)
10. Launch → `campaign-analytics` → measure → iterate
```

### Pattern B — Content Sprint
**ใช้เมื่อ:** Weekly content production
**Chain:**
```
1. Keyword brief from SEO team
2. `content-creator` → draft
3. `copywriting` → hook + CTA polish
4. `marketing-psychology` → emotional arc check
5. Publish + promote
6. `campaign-analytics` → track engagement
```

### Pattern C — Underperforming Channel Fix
**ใช้เมื่อ:** Channel ROI drops below target
**Chain:**
```
1. `campaign-analytics` → diagnose (reach? conversion? bid?)
2. `competitor-alternatives` → what's working elsewhere
3. `paid-ads` → refresh creatives + audiences
4. `copywriting` → new hooks
5. Small budget test → measure → scale or kill
6. `decision-logger` → channel decisions
```

## 🎯 How to Use

1. **แต่ละคน** → copy ไฟล์ที่ตรง role → เก็บไว้ที่ `02 - Areas/Team/Playbook - <ชื่อจริง>.md`
2. **กรอก:** ชื่อจริง, start date, manager (ใช้เวลา 5-10 นาที)
3. **Review ทุก 2 สัปดาห์** — skill ที่เขียนไว้ใช้จริงไหม ตัดอันที่ไม่ใช้ เพิ่มอันที่เพิ่งค้นพบ
4. **Share กับ manager** เพื่อ alignment

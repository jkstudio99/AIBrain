# 🕸️ Find Connections

## Usage
Discover hidden links and relationships between notes.

## Prompt
```
Find connections between these notes:

Note 1: [TITLE + SUMMARY]
Note 2: [TITLE + SUMMARY]
Note 3: [TITLE + SUMMARY]

Please:
1. Identify shared themes, concepts, or principles
2. Map how each note relates to the others (supports, contradicts, extends)
3. Suggest new notes that would bridge gaps between them
4. Recommend which MOC these belong to
5. Generate wikilinks in [[note title]] format
```

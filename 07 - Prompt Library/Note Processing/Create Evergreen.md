# 🌳 Create Evergreen Note

## Usage
Convert a fleeting or literature note into an atomic evergreen note.

## Prompt
```
Convert this note into an evergreen note:

[PASTE NOTE CONTENT]

Requirements:
1. Distill to ONE atomic idea (one idea per note)
2. Write a clear, declarative title (a statement, not a topic)
3. Express the idea in your own words (not quotes)
4. Keep it concise but complete
5. Suggest 3-5 connections to other potential notes
6. Add appropriate tags: #type/evergreen #status/seedling

Use this frontmatter:
---
type: evergreen
created: YYYY-MM-DD
status: seedling
tags: [type/evergreen, status/seedling]
related: []
---
```

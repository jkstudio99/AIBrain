# 💎 Extract Insights

## Usage
Extract key insights from raw notes, articles, or captured content.

## Prompt
```
Extract insights from this content:

[PASTE CONTENT]

Please provide:
1. **Key Insights** (3-5 atomic ideas, each one sentence)
2. **Surprising Elements** (what's unexpected or counterintuitive)
3. **Action Items** (what to do with this information)
4. **Connections** (what existing knowledge does this relate to)
5. **Questions** (what does this make me want to explore)

Format each insight as a potential [[evergreen note]] title.
```

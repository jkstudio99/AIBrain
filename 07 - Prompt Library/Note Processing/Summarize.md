# 📋 Summarize

## Usage
Create concise, useful summaries of long content.

## Prompt
```
Summarize this content at three levels:

[PASTE CONTENT]

1. **One sentence**: The core message
2. **One paragraph**: Key points and context
3. **Structured summary**:
   - Main argument/thesis
   - Supporting points (bulleted)
   - Key evidence/examples
   - Implications
   - My potential takeaways

Preserve any specific data, names, or references worth noting.
```

# 🧬 Synthesize — Idea Fusion

## Usage
Use to combine multiple ideas, notes, or sources into new insights.

## Prompt
```
Synthesize these ideas into something new:

Idea 1: [PASTE]
Idea 2: [PASTE]
Idea 3: [PASTE]

Please:
1. Find the common thread connecting these ideas
2. Identify tensions or contradictions between them
3. Create a new synthesis that integrates the best of each
4. Express this synthesis as a single clear statement
5. Suggest what this new idea implies or enables
6. Propose an evergreen note title for this synthesis
```

## When to Use
- Multiple notes on related topics
- Connecting ideas across domains
- Creating evergreen notes from literature notes
- Finding patterns across research

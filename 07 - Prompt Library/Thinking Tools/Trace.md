# 🔍 Trace — Deep Analysis

## Usage
Use when you need to trace through a complex idea, problem, or argument step by step.

## Prompt
```
I want to trace through this idea step by step:

[PASTE IDEA/PROBLEM HERE]

Please:
1. Break this into its fundamental components
2. Trace the logic chain from premise to conclusion
3. Identify any hidden assumptions
4. Note where the reasoning is strong vs weak
5. Suggest what's missing from the analysis

Format as a numbered reasoning chain with confidence levels.
```

## When to Use
- Complex problems requiring systematic analysis
- Debugging flawed reasoning
- Understanding multi-step processes
- Evaluating arguments or proposals

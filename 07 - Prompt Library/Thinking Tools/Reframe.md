# 🔄 Reframe — Perspective Shift

## Usage
Use when stuck or when a problem needs a fresh angle.

## Prompt
```
Help me reframe this situation from multiple perspectives:

[PASTE SITUATION HERE]

Please provide:
1. **Inversion**: What if the opposite were true?
2. **Zoom Out**: What does this look like from 10,000 feet?
3. **Zoom In**: What specific detail am I overlooking?
4. **Time Shift**: How will this matter in 5 years? How would I have seen this 5 years ago?
5. **Role Shift**: How would [expert/beginner/outsider] see this?
6. **Synthesis**: What new insight emerges from these perspectives?
```

## When to Use
- Feeling stuck on a problem
- Need creative breakthrough
- Tunnel vision on one approach
- Want to understand other viewpoints

# ⚔️ Challenge — Assumption Breaker

## Usage
Use to stress-test ideas, find blind spots, and strengthen arguments.

## Prompt
```
Challenge this idea rigorously:

[PASTE IDEA HERE]

Please:
1. Identify the top 3 assumptions being made
2. For each assumption, explain how it could be wrong
3. Present the strongest counter-argument
4. Find the weakest point in the reasoning
5. Suggest how to strengthen the idea against these challenges

Be constructively critical — the goal is to make this idea stronger.
```

## When to Use
- Before committing to a decision
- When an idea feels "too easy"
- Reviewing your own proposals
- Pre-mortem analysis
